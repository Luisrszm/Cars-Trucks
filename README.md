# Vehicle Management Command-Line Application

## Description

This command-line application allows developers to manage vehicles by creating new ones or using existing ones. It supports multiple vehicle types and enables users to perform various actions with the vehicles. The application is designed to help developers comprehend and work with existing codebases while extending functionality for new vehicle types.

## Installation

1. Clone the repository:  

   ```bash
   git clone https://github.com/Luisrszm/Cars-Trucks.git
   ```
2. Navigate to the project directory:  
   ```bash
   cd root/Cars-Trucks
   ```
3. Install required dependencies (if any):  
   ```bash
   npm install

   npm install inquirer
   ```

## Usage

Follow [this link](https://app.screencastify.com/v3/watch/awodwGFfnWDX9ZULlNRr) to watch a tutorial.

### Written instructions

1. **Run the Application**  
   Launch the command-line application.

2. **Choose an Option**  
   - Create a new vehicle.  
   - Select an existing vehicle.  

3. **Create a New Vehicle**  
   - Choose the type of vehicle: Car, Truck, or Motorbike.  
   - Enter the required details about the vehicle.  
   - Use the newly created vehicle to perform actions.  

4. **Use an Existing Vehicle**  
   - Select a vehicle from the list of existing vehicles.  
   - Perform actions with the selected vehicle.  

5. **Perform Actions**  
   - Interact with the vehicle and observe the results.  
   - Continue performing actions until you decide to exit.  

## How to run

Run the application using the following command:  
```bash
npm run start
```
